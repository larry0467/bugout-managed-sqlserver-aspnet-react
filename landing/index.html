How it works

Three steps to better bug reports.

1

Install the Widget

npm install @bugoutmanaged/widget and drop the component into your React app with your API key.

2

Users Report Issues

A floating orb appears in your app. Users click it, record their screen, describe the issue, and submit.

3

You Fix Faster

View recordings, transcripts, and full context in the admin dashboard. Triage, assign, and resolve.
